Deutsch-Daily is a vibe coding experiment using Claude to create the whole thing as a Progressive Web App (PWA).
The initial app was done on an iPhone in around 90 minutes using Claude and Github to serve the pages in an airline lounge in MSP (Minnesota) while waiting for my flight!

Each lesson consists of two nouns, a verb, an adjective and two sentences to illustrate those items.

Each Common European Framework of Reference for Languages (CEFR) level, A1, A2 etc. is presented as a set of 25 lessons grouped into blocks of five. 
After each block of 5 lessons there is a short test consisting of twelve questions on the contents of the 5 lessons in that block. 
The level test is broken down into:

3 noun translations
2 verb translations
1 adjective translation
2 sentence translations (German → English)
4 article questions (der / die / das)

Pass rate is 90% so a you have to get 11 or 12 correct to pass.

After the last test of the level is completed, there is a giant test of 40 questions on all the lessons in the level.
The final level exam is 40 questions:

14 noun translations
8 verb translations
6 adjective translations
6 sentence translations
6 article questions

All drawn from across all 25 lessons in the level. Both tests require 90% to pass.
After that you can procedd to the next CEFR level.
